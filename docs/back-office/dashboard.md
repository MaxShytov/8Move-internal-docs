# Дашборд

Дашборд — главная страница приложения, которая открывается сразу после входа в систему. Здесь собраны ключевые показатели и быстрые действия для повседневной работы.

---

## Быстрые действия

В верхней части дашборда расположена панель с 8 кнопками для быстрого доступа к основным операциям:

| Кнопка | Действие | Куда ведёт |
|--------|----------|------------|
| :material-cart-plus: **New Order** | Создать новый заказ | Форма создания заказа |
| :material-plus-box: **New Product** | Добавить новый продукт | Форма создания продукта |
| :material-account-plus: **New Client** | Создать нового клиента | Форма создания клиента |
| :material-account-multiple-plus: **New User** | Добавить нового пользователя | Форма создания пользователя |
| :material-format-list-bulleted: **View All Orders** | Все заказы | Список заказов |
| :material-package-variant: **View All Products** | Все продукты | Каталог продуктов |
| :material-domain: **All Clients** | Все клиенты | Список клиентов |
| :material-account-multiple: **All Users** | Все пользователи | Список пользователей |

Количество кнопок в ряду адаптируется под ширину экрана (от 2 до 8 в ряду).

## Статистика заказов

Карточка **Orders Statistics** показывает три показателя:

| Показатель | Описание | Индикатор тренда |
|------------|----------|-----------------|
| **Total Orders** | Общее количество заказов | — |
| **Pending Orders** | Заказы, не привязанные к маршруту | % от общего числа. Зелёная стрелка вверх :material-trending-up:, если менее 20% заказов в ожидании (хорошо). Красная стрелка вниз :material-trending-down: в обратном случае |
| **Completed Orders** | Завершённые заказы (общее минус ожидающие) | % от общего числа. Всегда зелёная стрелка :material-trending-up: |

Нажатие на любую карточку открывает [список заказов](orders.md).

---

## Статистика продуктов

Карточка **Products Statistics** показывает три показателя:

| Показатель | Описание | Индикатор тренда |
|------------|----------|-----------------|
| **Total Products** | Общее количество продуктов | — |
| **Active Products** | Продукты со статусом «активный» | % от общего числа. Зелёная стрелка :material-trending-up: |
| **Inactive Products** | Неактивные продукты | % от общего числа. Красная стрелка :material-trending-down: |

Нажатие на любую карточку открывает [каталог продуктов](products.md).

## Последние обновления

В правой части дашборда (или ниже на узких экранах) расположен виджет **Recent Updates** — лента последних изменений в приложении.

Показываются только **важные** обновления за последние 7 дат. Каждое обновление содержит:

- **Дату** выпуска
- **Заголовок** изменения
- **Описание**
- **Тип**: :material-star: новая функция (feature) или :material-bug: исправление (fix)

Для обновления ленты нажмите кнопку :material-refresh: в заголовке виджета.
