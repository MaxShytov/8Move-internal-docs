# Back Office — Руководство пользователя

## О приложении

**8Move Back Office** — веб-приложение для управления поставками, продуктами, клиентами и заказами. Предназначено для сотрудников компании-поставщика (supplier), которые управляют бизнес-процессами доставки.

## Возможности приложения

| Модуль | Описание |
|--------|----------|
| :material-view-dashboard: [Дашборд](dashboard.md) | Обзор ключевых показателей, быстрые действия |
| :material-cart: [Заказы](orders.md) | Создание, редактирование, управление статусами заказов |
| :material-receipt: [Счета](invoices.md) | Формирование счетов из доставленных заказов |
| :material-package-variant: [Продукты](products.md) | Каталог продуктов, ценообразование, категории |
| :material-account-group: [Клиенты](clients.md) | Управление клиентами, адресами, контактами |
| :material-account: [Пользователи](users.md) | Управление учётными записями |
| :material-cog: [Настройки поставщика](settings.md) | Конфигурация поставщика: валюта, налоги, выставление счетов |

## Быстрый старт

Если вы используете Back Office впервые, начните с раздела [«Быстрый старт»](quick-start.md), который описывает:

- Вход в систему и регистрацию
- Обзор интерфейса
- Навигацию по приложению
- Настройку профиля

## Поддерживаемые языки

Интерфейс приложения доступен на 4 языках:

- :flag_gb: English (по умолчанию)
- :flag_de: Deutsch
- :flag_fr: Français
- :flag_it: Italiano

Язык можно изменить в разделе [Профиль](quick-start.md#profile).

## Адаптивный интерфейс

- **Десктоп** (ширина экрана ≥ 1024px) — постоянная боковая панель навигации + верхняя панель
- **Планшет / Мобильный** (< 1024px) — выдвижное меню (гамбургер) + верхняя панель
