# Flutter Apps

## Applications

The project includes several Flutter applications:

### Back Office

Desktop application for internal management.

- **Location:** `back_office/`
- **Purpose:** Management dashboard, data entry, reports

### DocuFlow

Document processing application.

- **Location:** `docuflow/`
- **Purpose:** Document scanning, receipt processing

### Mobile App

Client-facing mobile application.

- **Location:** `mobile/` *(planned)*
- **Purpose:** End-user mobile experience
