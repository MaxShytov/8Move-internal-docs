    Быстрый старт¶

    Вход в систему¶

    Для входа откройте приложение Back Office в браузере. Вы увидите экран авторизации.

    Поля формы¶

    Поле Описание Обязательное
    Email Ваш email-адрес, указанный при регистрации
    Password Пароль от учётной записи

    Действия¶

    • Sign In — войти в систему
    • Forgot Password? — восстановить пароль (см. Восстановление пароля)
    • Sign Up — создать новую учётную запись (см. Регистрация)

    Подсказка

    Если вы видите красный баннер с ошибкой — проверьте правильность email и пароля.

    Регистрация¶

    Для создания новой учётной записи нажмите Sign Up на экране входа.

    Поля формы¶

    Поле Описание Обязательное Валидация
    First Name Имя Не может быть пустым
    Last Name Фамилия Не может быть пустым
    Email Email-адрес Должен содержать @
    Password Пароль Минимум 8 символов
    Confirm Password Повтор пароля Должен совпадать с Password
    Address Адрес — Необязательное поле

    После успешной регистрации вы будете перенаправлены на экран подтверждения email.

    Подтверждение Email¶

    После регистрации на ваш email будет отправлено письмо с ссылкой для активации аккаунта.

    Порядок действий:

    1. Откройте ваш почтовый ящик
    2. Найдите письмо с ссылкой активации
    3. Нажмите на ссылку в письме
    4. Вернитесь в приложение и войдите

    Не получили письмо?

    Проверьте папку «Спам» вашего почтового ящика.

    Восстановление пароля¶

    Если вы забыли пароль:

    1. Нажмите Forgot Password? на экране входа
    2. Введите ваш Email
    3. Нажмите Send Reset Link
    4. Проверьте почту — вам придёт письмо со ссылкой для сброса пароля
    5. Перейдите по ссылке из письма
    6. Введите новый пароль (минимум 8 символов) и подтвердите его
    7. Нажмите Reset Password

    После успешного сброса вы можете войти с новым паролем.

    Ссылка для сброса

    Ссылка для сброса пароля действительна ограниченное время. Если она истекла, вы увидите сообщение «Invalid Reset Link» — в этом случае запросите новую ссылку.

    Обзор интерфейса¶

    Верхняя панель (десктоп)¶

    Верхняя панель содержит:

    Элемент Описание
    Логотип Логотип текущего поставщика (или логотип 8Move по умолчанию)
    Название поставщика Название текущей компании + надпись «Back Office»
    Выбор поставщика Выпадающий список для переключения между поставщиками (только в мультипоставщическом режиме)
    Аватар пользователя Нажмите для перехода в Профиль

    Боковая панель навигации¶

    Боковая панель (или выдвижное меню на мобильных устройствах) содержит пункты:

    Иконка Пункт меню Описание
    Dashboard Главная страница с обзором
    Orders Управление заказами
    Invoices Счета
    Products Каталог продуктов
    Clients Управление клиентами
    Supplier Users Пользователи поставщика
    Settings Настройки поставщика
    Users Управление пользователями
    Help Помощь (в разработке)

    Текущий активный пункт меню выделен цветом.

    В нижней части панели отображается версия приложения.

    Профиль¶

    Для перехода в профиль нажмите на ваш аватар в правом верхнем углу интерфейса.

    Личные данные¶

    Поле Описание
    Аватар Фотография профиля (загрузка / удаление)
    First Name Имя
    Last Name Фамилия

    Язык интерфейса¶

    Вы можете выбрать один из 4 языков:

    • EN — English
    • DE — Deutsch
    • FR — Français
    • IT — Italiano

    Язык меняется мгновенно после выбора и сохраняется для будущих сессий. Выбранный язык также используется для запросов к серверу (все ответы API будут на выбранном языке).

    Смена пароля¶

    Для смены пароля:

    1. Введите текущий пароль
    2. Введите новый пароль
    3. Подтвердите новый пароль
    4. Нажмите кнопку сохранения

    Выход из системы¶

    Нажмите кнопку Logout в нижней части экрана профиля.

    Мультипоставщический режим¶

    Если ваша учётная запись привязана к нескольким поставщикам, в верхней панели появится выпадающий список для переключения между ними.

    • Текущий поставщик отмечен галочкой
    • При переключении все данные в таблицах обновляются для выбранного поставщика
    • Выбор сохраняется между сессиями

    Администратор (Superuser)

    Пользователи с правами суперпользователя видят дополнительную иконку в верхней панели. Через неё можно принудительно включить или выключить мультипоставщический режим.
